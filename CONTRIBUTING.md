# Contribuindo com o Projeto

Obrigado pelo interesse em contribuir com esta pesquisa! 🎉

Este projeto é uma pesquisa acadêmica aberta sobre **Text-to-Speech (TTS) com modelos open source em português brasileiro**, voltada à acessibilidade de pessoas com deficiência visual.

---

## Como Contribuir

### 1. Reportar Problemas (Issues)

Se você encontrou um bug, comportamento inesperado ou tem uma sugestão:

1. Acesse a aba [Issues](https://github.com/renatoork/tts-acessibilidade-pt-br/issues)
2. Clique em **New Issue**
3. Descreva o problema com o máximo de detalhes:
   - Ambiente (Colab, local, sistema operacional)
   - Passos para reproduzir
   - Comportamento esperado vs. observado
   - Logs de erro, se houver

### 2. Sugerir Melhorias

- Novos modelos TTS a serem avaliados
- Novas categorias de frases para o corpus
- Novas métricas de avaliação
- Melhorias na documentação
- Correções de erros no notebook

### 3. Contribuir com Código

1. **Fork** o repositório
2. Crie um branch para sua feature:
   ```bash
   git checkout -b feature/minha-contribuicao
   ```
3. Faça suas alterações e commit:
   ```bash
   git commit -m "feat: descrição da minha contribuição"
   ```
4. Envie para o seu fork:
   ```bash
   git push origin feature/minha-contribuicao
   ```
5. Abra um **Pull Request** descrevendo as mudanças

## Padrões de Código

- **Idioma**: todo o conteúdo textual deve estar em **português brasileiro**
- **Notebooks**: mantenha células bem comentadas e com outputs limpos antes do commit
- **Commits**: use mensagens descritivas em português ou inglês
